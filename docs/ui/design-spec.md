# UI 设计约定

适用于本项目所有小程序和 H5 Demo 页面的核心视觉约定。

## 色彩

**主色**：各项目自定义（如食物记录用金黄 `#e5a00d`，通用型用蓝色 `#2563eb`）

**语义色（统一）**：

- 成功：`#22c55e`
- 警告：`#f59e0b`
- 危险/错误：`#ef4444`

**文本色**：

- 主文本：`#2a2a2a`
- 次要文本：`#666666`
- 辅助/占位文本：`#999999`

## 间距

以 **4px** 为基础单位：


| 名称  | 值    |
| --- | ---- |
| 紧凑  | 8px  |
| 中等  | 12px |
| 标准  | 16px |
| 宽松  | 24px |
| 超宽  | 32px |


## 字号


| 用途   | 大小   |
| ---- | ---- |
| 辅助说明 | 12px |
| 正文   | 14px |
| 大号正文 | 16px |
| 小标题  | 18px |
| 大标题  | 24px |


## 圆角


| 场景      | 值      |
| ------- | ------ |
| 小标签、小按钮 | 6px    |
| 卡片、输入框  | 10px   |
| 大卡片     | 16px   |
| 胶囊形     | 9999px |


## 组件约定

- **可点击区域**最小高度：44px（满足移动端触控目标）
- **H5 页面**最大宽度：480px，居中显示
- **阴影**：慎用，仅用于浮层、FAB、主要操作按钮
