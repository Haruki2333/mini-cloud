# UI 设计规范

## 文档定位

本文件是项目 UI 设计的**入口与硬约束层**，分两部分：

1. **跨项目强制约束**：所有 H5 Demo 和小程序必须遵守的规则，无论各自的视觉主题如何
2. **项目设计索引**：指向各项目独立的视觉设计文档

各项目的色彩、字体、组件风格等**主题级决策**均在各自的设计文档中定义，本文件不重复也不覆盖它们。

---

## 跨项目强制约束

### 布局

- **H5 页面**：最大宽度 480px，水平居中，纵向 flex 布局，高度 `100dvh`
- **小程序页面**：遵循微信小程序官方布局规范

### 交互

- **可点击区域**最小高度：44px（满足移动端触控目标）
- **阴影**：慎用，仅用于浮层、弹窗、固定操作按钮；普通卡片避免使用

### 语义色（所有项目统一，不得覆盖）

| 语义 | 色值 |
|------|------|
| 成功 | `#22c55e` |
| 警告 | `#f59e0b` |
| 危险/错误 | `#ef4444` |
