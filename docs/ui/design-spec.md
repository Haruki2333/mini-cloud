# UI 设计约定

适用于本项目所有小程序和 H5 Demo 页面的核心视觉约定。

## 色彩

**主色**：各项目自定义（如食物记录用金黄 `#e5a00d`，通用型用蓝色 `#2563eb`）

**语义色（统一）**：
- 成功：`#22c55e`
- 警告：`#f59e0b`
- 危险/错误：`#ef4444`

**文本色**：
- 主文本：`#2a2a2a`
- 次要文本：`#666666`
- 辅助/占位文本：`#999999`

## 间距

以 **4px** 为基础单位：

| 名称 | 值 |
|------|-----|
| 紧凑 | 8px |
| 中等 | 12px |
| 标准 | 16px |
| 宽松 | 24px |
| 超宽 | 32px |

## 字号

| 用途 | 大小 |
|------|------|
| 辅助说明 | 12px |
| 正文 | 14px |
| 大号正文 | 16px |
| 小标题 | 18px |
| 大标题 | 24px |

## 圆角

| 场景 | 值 |
|------|-----|
| 小标签、小按钮 | 6px |
| 卡片、输入框 | 10px |
| 大卡片 | 16px |
| 胶囊形 | 9999px |

## 组件约定

- **可点击区域**最小高度：44px（满足移动端触控目标）
- **H5 页面**最大宽度：480px，居中显示
- **阴影**：慎用，仅用于浮层、FAB、主要操作按钮

## 食物记录 Demo — 光明通讯终端主题

食物记录 Demo 采用「光明通讯终端」视觉主题，营造与来自未来的仁慈 AI 通讯的氛围。

### 配色

| 用途 | 色值 | 说明 |
|------|------|------|
| 页面背景 | `#f8f7f4` | 温暖米白 |
| 卡片背景 | `#ffffff` | 纯白 |
| 主操作色 | `#e5a00d` | 金黄，用于按钮、光标、标记线 |
| 氛围渲染色 | `#e8a0b0` / `#fff0f0` | 浅粉，用于光晕、终端微光边框 |
| 主文本 | `#2a2a2a` | 深灰 |
| 系统消息 | `#999` | 浅灰 |
| 基础边框 | `#eee` | 极浅灰 |

### 字体

- **终端/标题/系统消息**：Space Mono（等宽字体）
- **中文正文**：Noto Sans SC

### 关键视觉效果

- 柔和光晕叠加层（浅粉色 radial-gradient 微光）
- 缓慢移动的粉色光带（梦幻氛围）
- 信号状态指示器（三格信号条 + 呼吸脉冲，金黄色）
- 终端对话框（白色卡片 + 浅粉色微光边框 + 消息逐条出现 + AI 逐字打出）
- 页面入场动画（柔和 fadeIn）
- 识别加载：信号环（粉色 + 金黄双色旋转）+ 渐变进度条 + 文字轮换

## 财务助理 Demo — 光明通讯终端财务模式

财务助理 Demo 复用光明通讯终端主题，聚焦文字记账与收支查询交互。

### 配色

与其他终端 Demo 一致：

| 用途 | 色值 | 说明 |
|------|------|------|
| 页面背景 | `#f8f7f4` | 温暖米白 |
| 次级背景 | `#f0efec` | 略深米白，用于悬停底色 |
| 卡片背景 | `#ffffff` | 纯白 |
| 主操作色 | `#e5a00d` | 金黄，用于按钮、光标、信号条、终端圆点 |
| 主操作浅色 | `rgba(229,160,13,0.12)` | 金黄透明，用于可点击项激活背景 |
| 氛围渲染色 | `#e8a0b0` / `#fff0f0` | 浅粉，用于右上角光晕 |
| 主文本 | `#2a2a2a` | 深灰 |
| 次要文本 | `#666666` | 中灰 |
| 辅助/占位文本 | `#999999` | 浅灰 |
| 基础边框 | `#eee` | 极浅灰 |
| 危险色 | `#ef4444` | 红色，用于删除、录音中指示、未配置提示红点 |
| 成功色 | `#22c55e` | 绿色 |

### 字体

- **终端标识、面板头、系统提示**：Space Mono（等宽字体）
- **中文正文、金额、标签**：Noto Sans SC

### 页面构成

财务助理共三个页面，均为纵向 flex、`100dvh`、`max-width: 480px` 居中布局：

#### 1. 首页（index.html）

| 区域 | 说明 |
|------|------|
| 顶部栏 | 信号指示器（三格金黄信号条 + `CONNECTED` 文字）+ 右侧圆形个人资料入口（未配置 API Key 时显示红点） |
| 终端头 | 金黄圆点（呼吸光晕动效）+ `FINANCE ASSISTANT` 标识 + 频道号 `FIN-01` |
| 月份选择行 | `MONTH>` 等宽标签 + 原生 `<input type="month">` |
| 月度摘要面板 | 终端面板（`MONTHLY_SUMMARY`），3 列网格：总支出 / 总收入 / 净收支；支出和收入格可点击（点击展开该月详细记录，激活态金黄下边线），净收支格为静态展示 |
| AI 回复面板 | 终端面板（`ASSISTANT_REPLY`），展示逐字打出的助理回复；打字中左侧显示 3px 金黄边线；空态显示 `> 等待输入...` |
| 固定底部输入栏 | 全宽渐变遮罩 + 文字输入框 + 金黄发送按钮（纸飞机图标） |

#### 2. 收支详情页（detail.html）

| 区域 | 说明 |
|------|------|
| 终端导航栏 | 返回箭头 + 页面标题（`EXPENSE_RECORDS` / `INCOME_RECORDS`） |
| 月份选择行 | 同首页 |
| 类型切换 Tabs | 全部 / 支出 / 收入，激活态金黄下划线 |
| 搜索行 | `SEARCH>` 等宽标签 + 文字过滤输入框 |
| 月度汇总面板 | 终端面板（`MONTHLY_TOTAL`），显示总金额 + 记录条数 |
| 记录列表面板 | 终端面板（`RECORD_LIST`），逐条展示流水；空态显示 `> 暂无记录` |

#### 3. 个人资料页（profile.html）

终端面板风格，依次包含：

| 面板标识 | 内容 |
|----------|------|
| `PERSONAL_INFO` | 称呼输入框 |
| `BUDGET_CONFIG` | 月预算（元）数字输入框 |
| `API_CREDENTIALS` | 千问 API Key（必填）+ 智谱 API Key（可选），均为密码型输入 |
| `AI_MODEL` | 模型选择按钮组 |

底部操作：「保存设置」主按钮 + 「清除聊天记录」危险轮廓按钮。

### 关键视觉效果

- **双色氛围光晕**：右上角浅粉 radial-gradient + 左下角金黄 radial-gradient，`position: fixed` 叠底，不影响交互
- **金黄信号条呼吸动效**（`signalPulse`）：三格信号条交替透明度 1↔0.5，间隔 0.2s
- **终端圆点光晕动效**（`dotGlow`）：金黄 box-shadow 0 0 4px ↔ 8px 循环
- **AI 打字动效**：回复逐字追加，打字期间左侧显示 3px 金黄实线
- **摘要格点击反馈**：hover 金黄淡底，active 金黄下边线 + 轻微缩放（scale 0.96）
- **Toast 提示**：底部弹出，自动消失

### 组件规格

| 组件 | 规格 |
|------|------|
| 终端面板（`.terminal-panel`） | 白色卡片，`border-radius: 12px`，`border: 1px solid #eee`，面板头等宽字体大写、字符间距 1px |
| 输入框（`.chat-input` / `.form-input`） | 圆角 8px，边框 1px `#eee`，focus 态金黄边框 + 金黄淡底色 |
| 发送按钮（`.send-btn`） | 40×40px 圆形，金黄背景，白色图标，active 缩放 0.92 |
| 主按钮（`.btn-primary`） | 全宽，金黄背景，白色文字，圆角 9999px，高度 48px |
| 危险轮廓按钮（`.btn-danger-outline`） | 全宽，透明背景，红色边框与文字，圆角 9999px |
| 类型 Tab（`.type-tab`） | 胶囊形，激活态金黄背景白色文字，非激活态透明 |

## 生活助理 Demo — 光明通讯终端对话模式

生活助理 Demo 复用光明通讯终端主题，聚焦语音对话交互。

### 布局

- 全屏 flex 纵向布局（100dvh），聊天框占据页面主体
- 顶部栏：信号指示器 + 个人资料入口（右上角人物图标，未配置 API Key 时显示红点）
- 终端头：LUMINA ASSISTANT CH-07
- 聊天消息区：flex:1 占满剩余空间，消息分左右对齐
- 底部：提示文字 + 72px 圆形麦克风按钮

### 交互

- 点击麦克风开始录音，按钮变为红色脉冲动画态
- 录音中聊天框底部显示临时气泡，实时显示 ASR 转写
- 再次点击停止录音，转写文字固定为用户消息
- 助理回复：逐字打出动画，左对齐带金黄左边线
- 用户消息：右对齐，金黄色背景白色文字

### 个人资料页

终端面板风格，包含：个人信息、API Key 配置、模型选择、保存 + 清除聊天记录按钮
